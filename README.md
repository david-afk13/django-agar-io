
```
cd agar.io
pip install django
python manage.py runserver
```

Then open in browser http://localhost:8000/

---

Endpoints html:  
- / registration
- / login
- /game
- /best_players

Endpoints API:
- /api/return_best_players
- /api/register
- /api/login
- /api/player/<int:player_id>/update
- /api/player/<int:player_id>/logout
